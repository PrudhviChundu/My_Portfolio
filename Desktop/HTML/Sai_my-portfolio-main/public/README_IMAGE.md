# Profile Image Setup

## Quick Setup:
1. Save your profile image to this folder (`public/`)
2. Name it exactly: `profile.jpg`
3. Refresh your browser

## Supported Formats:
- JPG/JPEG (recommended)
- PNG
- WebP

## Recommended Size:
- Square format: 500x500px or larger
- File size: Under 500KB for best performance

The image will automatically appear in the hero section of your portfolio!
