## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Customization

All personal information is located in the component files. Update the following:

- **Hero.jsx** - Name, title, contact info
- **About.jsx** - Objective and stats
- **Experience.jsx** - Work experience
- **Skills.jsx** - Skills and technologies
- **Education.jsx** - Education history
- **Hobbies.jsx** - Hobbies and personal info
- **Contact.jsx** - Contact information
